# Console Bank Application

A simple **Java-based console banking system** that simulates essential banking operations such as account creation, deposits, withdrawals, and balance inquiries.

This project is designed to demonstrate **core Java programming**, **object-oriented design**, and **basic financial transaction handling** through a command-line interface.

---

##  Features

-  Create new bank accounts
-  Deposit funds
-  Withdraw funds
-  Check account balance
-  View account details
-  Input validation & basic error handling

---

##  Tech Stack

- **Language:** Java
- **Application Type:** Console / CLI
- **Paradigm:** Object-Oriented Programming (OOP)

### Concepts Demonstrated

- Classes & Objects
- Encapsulation
- Abstraction
- Control Flow
- Data Management
- Exception Handling

---

##  How to Run

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/pronitray/console-bank-application.git
```
### 2️⃣ Navigate to the Project

```bash
cd console-bank-application
```
### 3️⃣ Compile the Source Files
```bash
javac src/*.java
```
### 4️⃣ Run the Application
```bash
java src.Main
```
